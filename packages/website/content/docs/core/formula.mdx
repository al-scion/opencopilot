# Formula

Fabric provides custom Excel formulas that let you generate AI content directly in your spreadsheet.

### GENERATE.TEXT

Generate text using AI.

```
=FABRIC.GENERATE.TEXT(prompt, [model])
```

| Parameter | Required | Description |
|-----------|----------|-------------|
| `prompt` | Yes | The text prompt to send to the AI |
| `model` | No | The language model to use |

**Example:**

```
=FABRIC.GENERATE.TEXT("Write a tagline for a coffee shop")
```

### GENERATE.IMAGE

Generate an image using AI.

```
=FABRIC.GENERATE.IMAGE(prompt, [model])
```

| Parameter | Required | Description |
|-----------|----------|-------------|
| `prompt` | Yes | Description of the image to generate |
| `model` | No | The image model to use |

**Example:**

```
=FABRIC.GENERATE.IMAGE("A sunset over mountains")
```
